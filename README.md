# Analyze Messages / Conversations based on LangFuse Traces

# Getting started

```bash
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
pip show langfuse
cp .env.example .env
vi .env
vi main.py (Update messages_log_file and select function, that you want to be executed)
python3 main.py
```
